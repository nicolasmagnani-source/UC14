Aula Tabelas

Introdução às Tabelas em HTML

Tabelas são estruturas em HTML que permitem a organização de dados em linhas e colunas.
São comumnente utilizadas para exibir calendários, horários, listas de preços, etc.

Estrutura Básica de uma Tabela (tags)

  • table: Define o início e o fim da tablea
  • tr: Define uma linha na tabela
  • td: Define uma célula dentro de um alinha (dados)
  • th: Deifine uma célula de cabeçalho
